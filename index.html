Lexical Analysis

Nine transformation operators. ~12,000 classified verb senses. All data open for audit.

EO proposed that every verb in every language performs one of 9 transformation types, arranged as three triads. This site tests that claim empirically: ~12,000 verb senses classified by LLM, embedded, and measured. Explore whether the categories cluster mathematically or are just imposed.

Preliminary explorations, not conclusions. Break it if you can.

11,933 verb senses classified
9 operators in 3 triads
3 embedding modes tested
27 languages (crosslinguistic)

Operator Overview

Three triads, generative pattern: op₁ + op₂ → op₃. Click to explore.

Three triads: Existence (NUL, DES, INS) — whether things exist. Structure (SEG, CON, SYN) — how things arrange. Interpretation (ALT, SUP, REC) — how things are understood. In each triad, op₁ + op₂ → op₃. Click any card for examples.

Open Questions & Next Experiments

Real vulnerabilities and unknowns.

Known vulnerabilities and genuinely open questions. Each card is a place where the framework could break.

Is nine the right number?

No sharp elbow at k=9 in silhouette scores. Some operators may need merging or splitting.

LLM classifier bias

The LLM knew about the 9 operators. Mode B (definitions only) partially tests this — structure persists but weakens.

Cross-linguistic validation

27 languages show z-scores below random, but effect varies. Small corpora show weak signals.

Type-signature formalization

Can operators be defined as pure type signatures? Mode A uses them, but they were written by the same LLM.

Adversarial examples

Find verbs that break the framework. Boundary verbs in the confusion matrix are a starting point.

Non-verb tests

Do nouns, syntax, or narrative structures decompose the same way? Verbs are just the first test.

The INS problem

INS contains 63% of all verbs (7,528). Genuine prevalence or catch-all bin?

SUP's vanishing act

Only 19 verbs as SUP. Cross-linguistically 0%–2.5% — consistently tiny but nonzero.
