# EO Lexical Analysis — v16.3 Operator Definition Update Log

## Date: 2026-02-28

## What Changed

Based on Emergent Ontology Handbook PATCH v16.3, two operator definitions were corrected
to reflect biological grounding — ensuring the operators work at the cellular level,
not just the human-linguistic level.

### DES: Designation → Distinction

**Why:** "Designation" implies a namer — a conscious agent who chooses a label. But DES
operates before consciousness, before language, before choice. A bacterium does not
"designate" a sugar molecule. But when the molecule hits a receptor and the receptor
responds, something has become something rather than anything. A distinction has been
drawn where there wasn't one.

**Old definition:**
```
DES (⊡) — Designation
Core Action: Name, type, categorize, assign identity
short_def: "Carve a domain into stable types by naming, defining, and establishing constraints."
```

**New definition:**
```
DES (⊡) — Distinction
Core Action: Draw distinction / mark difference
What Transforms: Something registers as not-that
Biological ground: Receptor responds to molecule
short_def: "Draw a distinction — register something as different from its ground, marking what it is."
```

DES scales from receptor distinguishing molecule from medium, to infant distinguishing
mother's face from background, to philosopher carving a new conceptual category.
"Designation" captures only the top of this range. "Distinction" captures the full span.

### ALT: Frame-Switching → State Change

**Why:** ALT's core operation is not "shift the interpretive frame." It is simpler:
same entity, different state. A bacterium's flagellum spins one direction to swim
forward. It spins the opposite direction to tumble. Same motor, different state.
That is ALT. Frame-switching and perspective change are complex instances of ALT —
things ALT can do when applied at higher scales — but they are not the definition.

**Old definition:**
```
ALT (∿) — Alternation
Core Action: Switch perspective, reframe, shift interpretive frame
short_def: "Switch perspective, definition, or interpretive frame without changing underlying data."
```

**New definition:**
```
ALT (∿) — Alternation
Core Action: Change state / toggle
What Transforms: Same entity, different state
Biological ground: Flagellum reverses spin direction
short_def: "Change state — same entity, different state. Toggle between alternatives."
```

Reframing remains a valid instance of ALT — specifically, ALT aimed at Pattern-level
or Ground-level targets in the Interpretation domain. But it is one application,
not the definition.

## Expected Impact

The definition changes may affect verb classifications in two ways:

1. **DES boundary shifts:** Some verbs previously classified as DES (because they
   "name" or "type") might now be classified differently if the classifier interprets
   "draw distinction" differently from "assign type." Conversely, some verbs classified
   elsewhere might now be drawn toward DES if "distinction" is a better fit.

2. **ALT boundary shifts:** Some verbs classified as ALT (because they "reframe")
   might shift to other operators. And some verbs classified elsewhere (e.g., as INS
   for state changes that are really toggling) might now land on ALT. The ALT-SUP
   boundary should become clearer since ALT is now "state change" vs SUP's "holding
   contradictory states."
